# A股短线技术分析参考手册

## 一、量价关系分析

### 1.1 经典量价组合

| 组合 | 含义 | 短线信号 |
|------|------|----------|
| **放量上涨** | 资金积极做多，量增价涨 | ✅ 强烈看多 |
| **缩量上涨** | 上涨动力衰减，资金观望 | ⚠️ 谨慎追高 |
| **放量下跌** | 恐慌盘涌出，多空分歧加大 | ❌ 看空 |
| **缩量下跌** | 抛压减轻，可能接近底部 | ⚡ 可能止跌 |
| **天量天价** | 高位巨量换手，主力可能出货 | ❌ 高位风险 |
| **地量地价** | 交投清淡，底部特征 | ⚡ 可能反转 |
| **温和放量上涨** | 多头有序进攻 | ✅ 健康上涨 |

### 1.2 关键量能指标

- **量比 > 2.0**：明显放量，关注度提升
- **量比 > 5.0**：巨量异动，可能有重大消息
- **量比 < 0.5**：极度缩量，市场冷清
- **换手率 > 15%**：高换手，短线活跃
- **换手率 < 3%**：低换手，流动性差

### 1.3 短线量价实战要点

1. **底部放量突破**：长时间缩量横盘后突然放量突破，是经典启动信号
2. **缩量回调买入**：上升趋势中的缩量回踩支撑位，是买入良机
3. **高位放量大阴线**：顶部的巨量长阴线是坚决卖出信号
4. **量价背离**：价格新高但成交量萎缩，需警惕见顶

| 形态 | 特征 | 可靠性 |
|------|------|--------|
| **早晨之星** | 大阴线 + 小实体(十字星) + 大阳线 | ⭐⭐⭐⭐ |
| **刺透形态** | 大阴线后阳线深入阴线实体过半 | ⭐⭐⭐⭐ |
| **锤子线** | 下影线长度至少2倍实体 | ⭐⭐⭐ |

| **红三兵** | 三根连续小阳线，稳步上涨 | ⭐⭐⭐ |

## 三、均线系统分析

### 3.1 均线多头/空头排列

- **多头排列**：MA5 > MA10 > MA20 > MA60，各均线向上发散 → 强势看多
- **空头排列**：MA5 < MA10 < MA20 < MA60，各均线向下发散 → 强势看空
- **均线纠缠**：多条均线粘合在一起，即将变盘 → 关注突破方向

### 3.2 关键均线信号

| 信号 | 描述 | 操作 |
|------|------|------|
| **金叉** | 短期均线上穿长期均线 | 买入信号 |
| **死叉** | 短期均线下穿长期均线 | 卖出信号 |
| **回踩不破** | 价格回踩均线后反弹 | 买入机会 |
| **跌破支撑** | 价格跌破重要均线 | 止损信号 |
| **多头排列形成** | 均线系统转向多头 | 建仓信号 |

### 3.3 均线实战要点

1. **20日均线**：短期生命线，跌破需警惕
2. **60日均线**：中线牛熊分界线
3. **均线间距**：多头排列中均线间距过大，短期有回调压力
4. **成交量配合**：均线金叉配合放量，信号更可靠

## 四、MACD指标分析

### 4.1 MACD基础

- **DIF（快线）**：短期EMA与长期EMA的差值
- **DEA（慢线）**：DIF的平滑值
- **MACD柱（红绿柱）**：(DIF - DEA) × 2

### 4.2 关键信号

| 信号 | 描述 | 可靠性 |
|------|------|--------|
| **金叉** | DIF上穿DEA | 买入信号 |
| **死叉** | DIF下穿DEA | 卖出信号 |
| **零轴上方金叉** | DIF和DEA都在零轴上方时金叉 | ⭐⭐⭐⭐ 强买入 |
| **零轴下方死叉** | DIF和DEA都在零轴下方时死叉 | ⭐⭐⭐⭐ 强卖出 |
| **底背离** | 价格新低但MACD不再新低 | ⭐⭐⭐⭐⭐ 见底信号 |
| **顶背离** | 价格新高但MACD不再新高 | ⭐⭐⭐⭐⭐ 见顶信号 |
| **红柱缩短** | 上涨动能减弱 | ⚠️ 注意风险 |
| **绿柱缩短** | 下跌动能减弱 | ⚡ 可能反弹 |

### 4.3 MACD短线实战

1. **零轴上方运行**：整体处于多头区域，回调到零轴是买点
2. **二次金叉**：DIF回落到DEA附近再次金叉，是强买入信号
3. **底背离共振**：MACD底背离 + KDJ底背离 + 成交量放大 → 极强买入信号

## 五、KDJ指标分析

### 5.1 KDJ基础读数

- **K > 80，D > 80**：超买区域，注意回调风险
- **K < 20，D < 20**：超卖区域，关注反弹机会
- **J > 100**：极度超买，强烈卖出信号
- **J < 0**：极度超卖，强烈买入信号

### 5.2 关键信号

| 信号 | 描述 | 操作 |
|------|------|------|
| **金叉(K上穿D)** | K线从下方上穿D线 | 买入 |
| **死叉(K下穿D)** | K线从上方下穿D线 | 卖出 |
| **超买区金叉** | 80以上区域金叉 | 强势追涨（趋势股） |
| **超卖区金叉** | 20以下区域金叉 | 反弹买入 |
| **高位钝化** | KDJ在80以上长时间运行 | 趋势强劲，不急于卖出 |
| **低位钝化** | KDJ在20以下长时间运行 | 弱势行情，谨慎抄底 |

### 5.3 KDJ实战注意

- KDJ是**震荡指标**，在强趋势行情中容易反复金叉/死叉
- 最好配合MACD趋势指标使用：MACD确认方向 + KDJ找买卖点
- 日线KDJ金叉 + 60分钟KDJ金叉 = 更可靠的买入信号

## 六、布林带分析

### 6.1 布林带基础

- **上轨(UP)**：中轨 + 2倍标准差
- **中轨(MID)**：20日均线
- **下轨(DN)**：中轨 - 2倍标准差

### 6.2 关键信号

| 信号 | 描述 | 操作 |
|------|------|------|
| **触及上轨** | 价格触及或突破上轨 | 短线超买，注意回落 |
| **触及下轨** | 价格触及或跌破下轨 | 短线超卖，可能反弹 |
| **收口** | 上下轨距离缩小 | 变盘在即 |
| **开口** | 上下轨距离扩大 | 趋势启动，顺势而为 |
| **沿上轨运行** | 价格紧贴上轨上涨 | 强势特征 |
| **沿下轨运行** | 价格紧贴下轨下跌 | 弱势特征 |
| **突破上轨放量** | 带量突破上轨 | 加速上涨 |

## 七、短线选股策略

### 7.1 龙头战法（核心策略）

**选股条件：**
1. 板块涨幅前3名中的领涨股
2. 首板涨停或连板股（2-3板最佳）
3. 封板时间早（10:00前封板最好）
4. 封单量大（资金坚决做多）
5. 所属板块为当日主线

**操作规则：**
- 首板：确认板块强势后，次日低吸或竞价跟随
- 2板：确认龙头地位，可追涨
- 3板及以上：风险加大，只做龙头
- 断板：次日低开且不反包则止损

### 7.2 板块轮动策略

1. 每日查看板块涨幅排行，找到当日最强板块
2. 在最强板块中选择领涨股
3. 注意板块间的轮动规律（科技→消费→周期→金融）
4. 同一板块内做高低切换

### 7.3 回调买入策略

**适用条件：**
- 个股处于明确上升趋势（MA20向上，多头排列）
- 前期有放量涨停或大阳线（确立强势）
- 回调缩量（量能萎缩到放量时的一半以下）
- 回调到关键支撑位（MA10/MA20/前期平台/涨停价）

**买入信号：**
- 回调到支撑位出现止跌K线（长下影线、小阳线）
- 成交量缩量后重新放量
- MACD绿柱缩短或DIF即将金叉DEA

### 7.4 突破买入策略

**适用条件：**
- 个股横盘整理3-5天以上
- 横盘期间成交量明显萎缩
- 均线逐步粘合

**买入信号：**
- 放量突破横盘区间上沿
- 突破时MACD在零轴上方或即将金叉
- 板块配合（同板块有其他个股走强）

## 八、风险控制

### 8.1 止损纪律

| 情形 | 止损标准 |
|------|----------|
| **买入后亏损** | 单笔亏损不超过总资金的5% |
| **跌破关键位** | 跌破买入时依据的支撑位 |
| **趋势破坏** | 跌破MA20或均线死叉 |
| **大盘急跌** | 大盘跌破重要支撑时减仓 |

### 8.2 仓位管理

- **单只个股**：不超过总仓位的20%
- **单板块**：不超过总仓位的40%
- **总仓位**：大盘弱势不超过50%，强势可以80%
- **分批建仓**：首次半仓，确认后加仓

### 8.3 短线操作铁律

1. **不追高**：已经3个涨停以上的不追
2. **不抄底**：下跌趋势中的个股不碰
3. **不扛单**：到达止损位必须执行
4. **不频繁交易**：每天最多1-2笔交易
5. **顺势而为**：只做上升趋势中的个股

## 九、日内短线(T+0)参考

### 9.1 分时图关键时点

| 时间 | 关注要点 |
|------|----------|
| **09:25-09:30** | 集合竞价，看开盘价和量能 |
| **09:30-09:45** | 开盘15分钟，消化隔夜情绪 |
| **09:45-10:00** | 方向确认期，重要买卖窗口 |
| **10:00-10:30** | 主力资金活跃时段 |
| **10:30-11:30** | 趋势延续或反转 |
| **13:00-13:30** | 午后开盘方向 |
| **14:30-15:00** | 尾盘博弈，次日预期定价 |

### 9.2 分时图买卖信号

- **均价线突破**：价格突破分时均价线且放量 → 买入
- **均价线支撑**：价格回踩均价线后反弹 → 买入
- **量价齐升**：分时图上价格和成交量同步上升 → 持有
- **量价背离**：价格新高但成交量萎缩 → 减仓

## 十、综合评分体系

对个股进行1-10分的短线评分，综合考虑以下维度：

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| **趋势** | 25% | 均线排列、价格位置 |
| **量能** | 20% | 量比、换手率、量价关系 |
| **MACD** | 15% | 金叉/死叉、背离、零轴位置 |
| **KDJ** | 10% | 超买超卖、金叉死叉 |
| **K线形态** | 15% | 看涨/看跌组合、关键K线 |
| **板块** | 15% | 板块强度、龙头地位 |

**评分建议：**
- **8-10分**：强烈推荐，可重仓
- **6-7分**：可以关注，轻仓参与
- **4-5分**：中性，观望
- **1-3分**：回避
